安装：著名的五分钟安装

  1. 在一个空目录中解压包并上传所有内容。
  2. 在您的浏览器中打开 wp-admin/install.php。它将引导您完成设置一个包含数据库连接详情的 wp-config.php 文件的过程。
    1. 如果由于某种原因这不起作用，不用担心。它可能不适用于所有的网络主机。使用像记事本或类似的文本编辑器打开 wp-config-sample.php 并填写您的数据库连接信息。
    2. 将文件另存为 wp-config.php 并上传。
    3. 在您的浏览器中打开 wp-admin/install.php。
  3. 一旦配置文件设置完成，安装程序将设置您的站点所需的表。如果出现错误，请仔细检查您的 wp-config.php 文件，并再试一次。如果再次失败，请带着尽可能多的信息前往 WordPress 支持论坛。
  4. 如果您没有输入密码，请牢记生成的随机密码。 如果您没有提供用户名，它将是 admin。
  5. 随后安装程序会带您到 登录页面。使用您在安装过程中设置的用户名和密码登录。如果 WordPress 为您生成了密码，您可以点击「个人资料」来更改密码。
  6. （可选）我们为简体中文版提供了一些本地化特性，您可以通过文章 简体中文的本地化特性 了解它们。

更新

使用自动更新器

  1. 在您的浏览器中打开 wp-admin/update-core.php 并遵循指示。
  2. 您想要更多，也许？就是这样！

手动更新

  1. 在您更新任何东西之前，确保您有任何您可能修改过的文件的备份副本，如 index.php。
  2. 删除您的旧 WordPress 文件，保存您修改过的文件。
  3. 上传新文件。
  4. 在您的浏览器打开 /wp-admin/upgrade.php。

从其他系统迁移

WordPress 可以 从许多系统导入。首先您需要按照上述描述安装并运行 WordPress，然后使用 我们的导入工具。

系统要求

  • PHP 版本 7.0 或更高。
  • MySQL 版本 5.5.5 或更高。

推荐

  • PHP 版本 7.4 或更高。
  • MySQL 版本 8.0 或更高 或 MariaDB 版本 10.4 或更高。
  • mod_rewrite Apache 模块。
  • HTTPS 支持。
  • 在您的站点上放置一个指向 cn.wordpress.org 或 wordpress.org 的链接。

在线资源

如果您有任何未在本文档中解决的问题，请利用 WordPress 的众多在线资源：

HelpHub
HelpHub 是 WordPress 所有事物的百科全书。它是可用的最全面的 WordPress 信息源。
WordPress 新闻
这里您可以发现与 WordPress 相关的最新更新和新闻。最近的 WordPress 新闻默认会出现在您的管理仪表板中。
WordPress Planet
WordPress Planet 是一个新闻聚合器，汇集了来自网络上 WordPress 博客的帖子。
WordPress 支持论坛
如果您到处都找不到答案，支持论坛非常活跃并且拥有一个大型的社区准备帮助。为了帮助他们帮助您，请确保使用描述性的线索标题并尽可能详细地描述您的问题。
WordPress IRC（互联网中继聊天）频道
这是一个在线聊天频道，用于 WordPress 用户之间的讨论，偶尔也会讨论支持主题。上面的 wiki 页面应该会指引您正确的方向。（irc.libera.chat #wordpress）
